# balaji-scripts-collection
Python and PySpark scripts for data engineering, automation, and reusable ETL utilities.

This repository contains a growing collection of Python and PySpark scripts designed for data engineering workflows. It includes utilities for automation, Hadoop/Hive/Spark data processing, ETL pipeline support, DB operations, monitoring jobs, validation checks, and general-purpose scripting. The goal is to maintain reusable, well‑structured components for both development and production environments.
